# Configuration Reference

All Plexo configuration is done through environment variables in `.env`. The `install.sh` script generates required secrets automatically; everything else is optional.

## Required

These must be set before `docker compose up`. The install script generates them for you.

| Variable | Description | Generate with |
|----------|-------------|---------------|
| `POSTGRES_PASSWORD` | PostgreSQL superuser password | `openssl rand -hex 32` |
| `SESSION_SECRET` | Signs and encrypts user sessions | `openssl rand -hex 64` |
| `ENCRYPTION_SECRET` | AES-256-GCM key for encrypting API credentials at rest | `openssl rand -hex 32` |
| `PUBLIC_URL` | Full URL with protocol, e.g. `https://plexo.example.com` | Set manually |
| `PUBLIC_DOMAIN` | Domain without protocol, e.g. `plexo.example.com` | Set manually |

## AI Providers

API keys can be set here or configured in-app via **Settings > AI Providers**. Keys entered in the UI are encrypted at rest using `ENCRYPTION_SECRET`. Setting them in `.env` pre-seeds the provider so it's available immediately on first boot.

| Variable | Provider | Get key at |
|----------|----------|------------|
| `ANTHROPIC_API_KEY` | Anthropic (Claude) | [console.anthropic.com](https://console.anthropic.com/settings/keys) |
| `OPENAI_API_KEY` | OpenAI (GPT) | [platform.openai.com](https://platform.openai.com/api-keys) |
| `GEMINI_API_KEY` | Google Gemini | [aistudio.google.com](https://aistudio.google.com) |
| `GROQ_API_KEY` | Groq | [console.groq.com](https://console.groq.com) |
| `MISTRAL_API_KEY` | Mistral | [console.mistral.ai](https://console.mistral.ai) |

Ollama (local models) requires no API key -- configure it in-app with a base URL.

## Channel Integrations

Enable external messaging channels. All are optional.

### Telegram

| Variable | Description |
|----------|-------------|
| `TELEGRAM_BOT_TOKEN` | Bot token from [@BotFather](https://t.me/BotFather). After setting, configure the webhook: `${PUBLIC_URL}/api/telegram/webhook` |

### Slack

| Variable | Description |
|----------|-------------|
| `SLACK_CLIENT_ID` | OAuth client ID from [api.slack.com/apps](https://api.slack.com/apps) |
| `SLACK_CLIENT_SECRET` | OAuth client secret. Redirect URL: `${PUBLIC_URL}/api/oauth/slack/callback` |

### Discord

| Variable | Description |
|----------|-------------|
| `DISCORD_APP_ID` | Application ID from [discord.com/developers](https://discord.com/developers/applications) |
| `DISCORD_PUBLIC_KEY` | Interactions verification key |
| `DISCORD_BOT_TOKEN` | Bot token. Interactions endpoint: `${PUBLIC_URL}/api/discord/interactions` |

## OAuth Providers

Enable third-party login and service connections.

### GitHub

| Variable | Description |
|----------|-------------|
| `GITHUB_CLIENT_ID` | OAuth app ID. Create at [github.com/settings/applications/new](https://github.com/settings/applications/new) |
| `GITHUB_CLIENT_SECRET` | OAuth secret. Callback: `${PUBLIC_URL}/api/oauth/github/callback` |

### Google

| Variable | Description |
|----------|-------------|
| `GOOGLE_CLIENT_ID` | OAuth 2.0 client ID from [Google Cloud Console](https://console.cloud.google.com) > Credentials |
| `GOOGLE_CLIENT_SECRET` | OAuth secret. Callback: `${PUBLIC_URL}/api/oauth/google/callback` |

## Asset Storage

Plexo uses S3-compatible storage for agent-produced files. The Docker Compose stack includes MinIO by default.

| Variable | Default | Description |
|----------|---------|-------------|
| `STORAGE_ACCESS_KEY` | `plexo` | S3 access key (matches `MINIO_ROOT_USER`) |
| `STORAGE_SECRET_KEY` | -- | S3 secret key (matches `MINIO_ROOT_PASSWORD`). Generate: `openssl rand -hex 32` |
| `STORAGE_BUCKET` | `plexo-assets` | Bucket name |
| `STORAGE_ENDPOINT` | `http://minio:9000` | Override to use external S3, Cloudflare R2, or Backblaze B2 |

## Cost Controls

| Variable | Default | Description |
|----------|---------|-------------|
| `API_COST_CEILING_USD` | `10.00` | Weekly AI spend limit in USD. Tasks auto-pause when reached. |
| `MAX_SPRINT_WORKERS` | `5` | Max concurrent sprint workers. Reduce on memory-constrained VPS. |

## Voice

## Observability

| Variable | Description |
|----------|-------------|
| `SENTRY_DSN` | Sentry error reporting DSN |
| `SENTRY_WEBHOOK_SECRET` | Sentry webhook secret for task-on-issue integration |
| `POSTHOG_API_KEY` | PostHog analytics (opt-in) |

## Instance

| Variable | Description |
|----------|-------------|
| `PLEXO_INSTANCE_ID` | Unique instance identifier (auto-generated by `install.sh`) |
| `SUPER_ADMIN_EMAILS` | Comma-separated emails that get auto-promoted to super admin |

## Infrastructure Overrides

These are auto-configured by Docker Compose. Only set them if running services externally.

| Variable | Default (in Docker) | Description |
|----------|---------------------|-------------|
| `DATABASE_URL` | `postgresql://plexo:${POSTGRES_PASSWORD}@plexo-db:5432/plexo` | PostgreSQL connection string |
| `REDIS_URL` | `redis://redis:6379` | Valkey/Redis connection string |

## Docker-specific

| Variable | Default | Description |
|----------|---------|-------------|
| `DOCKER_SOCKET_ENABLED` | `true` | Enables one-click update via Docker API |
| `COMPOSE_PROJECT_NAME` | `plexo` | Docker Compose project name |
