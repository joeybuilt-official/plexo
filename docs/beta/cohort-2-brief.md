# Cohort 2 Brief — Broader Users

**Size**: 75 testers
**Engagement window**: 2 weeks from invite
**Prerequisite**: Cohort 1 findings triaged, all critical issues resolved
**Feedback deliverable**: Completed structured feedback form (see `feedback-form.md`)

## Selection Criteria

- Developers, indie builders, technical founders, small team leads
- Interested in agent-assisted work but not necessarily experienced self-hosters
- Mix of backgrounds: solo developers, startup CTOs, DevRel, technical PMs
- Comfortable using a web UI and connecting an AI provider API key

## Testing Brief

### 1. Get Plexo running

Either self-host using `docs/self-host.md` or use the hosted preview instance (link provided in invite).

### 2. Run at least 3 tasks of your choice

No category requirement. Use Plexo for whatever real work you have. The more authentic the task, the more useful the feedback.

### 3. Answer four questions after testing

1. **Did the agent do what you asked?** (Yes / Mostly / No)
2. **Did you understand what it produced?** (Yes / Mostly / No)
3. **Would you use this weekly?** (Yes / Maybe / No)
4. **What's the one thing that would make you recommend Plexo to someone else?**

## Feedback Deliverable

Complete the structured feedback form within the 2-week window. Include at least one specific task example with the prompt you used and what the agent produced.

## Gate Criteria

When the majority of Cohort 2 answers **Yes** to questions 1 and 2 ("Did the agent do what you asked?" and "Did you understand what it produced?"), the public announcement gate is unlocked.
